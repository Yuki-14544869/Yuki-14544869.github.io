#1327 : 分隔相同字符

描述

给定一个只包含小写字母’a’-‘z’的字符串 S ，你需要将 S 中的字符重新排序，使得任意两个相同的字符不连在一起。

如果有多个重排后字符串满足条件，输出字典序最小的一个。

如果不存在满足条件的字符串，输出INVALID。

输入

字符串S。(1 ≤ |S| ≤ 100000)

输出

输出字典序最小的答案或者INVALID。

样例输入

aaabc

样例输出

abaca

限制

时间限制:10000ms
单点时限:1000ms
内存限制:256MB

思路

贪心，由题意易知VALID的充要条件即为此字母的长度小于整个字符串长度len/2向上取整。在每次贪心之后重新check新字符串是否VALID就可。
